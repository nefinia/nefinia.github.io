PLOT ARMOR

A game about surviving absolutely impossible situations with explanations that should not work — but somehow do.

What is this?

In each round, a character is placed in a ridiculous, clearly deadly situation. Your job is to explain how they survive.

You are given a small hand of random cards. They don’t explain anything. They just exist.

You must force them to make sense.

What you need to play

  • 2–8 players
  • One shared screen or voice call
  • Each player opens user.html on their own device
  • Trust (this is a beta)

Game flow

  1. One player reads or generates the public situation:
    • Setting
    • Character
    • Threat
  2. Each player looks at their private hand in user.html
  3. Each player chooses one card to use
  4. Each player gives one sentence explaining how the character survives
  5. The group (or a director) decides the best explanation
  6. Winner gets a point, start a new round

Important rules

  • You always keep exactly 3 cards per deck
  • Once a card is used, it should not be reused
  • No meta explanations (“movie logic”, “cut to black”)
  • Short, confident explanations work best
  • This is a trust-based beta — don’t cheat
